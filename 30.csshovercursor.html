Css Hover

CSs hover is used to enhance th e user experience. We can change the appearance or the behaviour of the HTML elements when the user hover on it.

<------------------Syntax---------------->
 selector:hover{
      /* CSS rule(s) */;
  }
<-----------------Example---------------->

  <html lang="en">
    <head>
        <style>
            #p1{
                color: yellow;
            }
            #p1:hover{
                color: red;
            }
    
            #p2 {
                border: 2px solid red;
            }
            #p2:hover{
                border: 5px solid purple;
            }
        </style>
    </head>
    <body>
        <p id="p1">CSS Hover</p>
        <p id="p2">CSS hover is used to enhance the user experience.</p>
    </body>
    </html>

This is one example of using CSS hover; you can try many other things too.

Custom Cursor

With all the predefined cursors, we can create our own cursor.

  .selector {
      cursor: url('custom-cursor.png'), auto;
  }

The auto keyword ensures that the browser falls back to the default cursor when the custom cursor fails to load.
